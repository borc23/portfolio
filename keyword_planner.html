Keyword Planner

The project was intended to help with digital marketing agency to automate their SEO, keyword research and analyzation of future or past keyword metrics.

With the usage of different APIs such as OpenAI API, Google Ads API, Google Drive API, Google Sheets API I was able to just make that a reality.

I was able to shorten their research time from few hours to few minutes.

The program generated keywords based on the company and the product you are developing the keywords for with special knack of making them editable to your standards once they are generated.

For web development I used Flask as the program is for internal usage, while I also used languages HTML, CSS and JavaScript to be able to make it into a reality.

For backend I have used Python as I know how to use best practices to work with APIs limitations and able to "bend" the rules to make it work for my client's needs and wishes.

Critical thinking and analysis were able to make this project into a sufficient and well managed process.

Technologies:
  • - Python
  • - Flask
  • - HTML, CSS, JavaScript
  • - OpenAI API
  • - Google Ads API
  • - Google Drive API
